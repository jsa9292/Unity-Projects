[Demo Play Guide]
1. Menu Bar : File >> Build Settings >> Add Open Scenes (Demo, Launcher, Melee, Shooter, Total <<Scene in "Demo" folder)
2. Open Demo.scene in "Demo" folder.
3. Play.
4. "Choice Model" click the button (3 models change).
5. Click the "animation button" at the bottom.
6. Animations are played for about 3 seconds.
